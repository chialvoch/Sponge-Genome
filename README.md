# Genome Analysis Script for _Radiospongilla crateriformis_
This repository contains the custom bash scripts used for the _de novo_ genome assembly, annotation, structural analysis, and genomic feature characterization of _Radiospongilla crateriformis_.

__Code Written By:__ Summer Dieterle, Taneille Jordan, Clare Scott Chialvo

## Dependencies
The workflows in this repository utilizes standard bioinformatics tools



## Repository Contents
1. _de novo_ assembly with __flye__
2. Assembly polishing with long reads with __Racon__ (x3)
3. Quality check of assembly with __BUSCO__
4. Polishing the assembly with the short reads with __Pilon__
